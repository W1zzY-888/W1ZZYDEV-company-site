- После изменений обновлять версии `styles.css?v=...` и `script.js?v=...` в `index.html`, чтобы GitHub Pages не показывал старый кэш.

- Коммитить только относящиеся к задаче файлы.
- После push проверять, что GitHub Pages отдает новую версию.
